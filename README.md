# cobaltstrike_plugin
陆续补充一些自己写的cs插件

# 仅供交流与学习使用，请勿用于非法用途！

# 功能简介
sunlogin 向日葵识别码和密文的读取(老版本和新版本都可)
teamviewer 抓取ID和密码

向日葵，teamviewer，Todesk的应用弹起到桌面并保存截图
